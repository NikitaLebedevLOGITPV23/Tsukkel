welcome to

follow these steps to create a spatially enabled web application.

find Data
load file-based data

load geospatial files, such as SHP or SDF, to the Infrastructure Map Server Library. Save the load procedures to refresh the data.

connect to external data

configure connections to files such as SHP or SDF, databases such as Oracle, and web services such as WMS or WFS.

build Layers
create a new layer

linework in the source data is monochromatic by default. data needs to be linked to a layer for styling. you can theme by attributes, such as parcels colored by property value or road thickness indicating traffic volume.

make a Map
create a new map

now that you've themed your layers to give them meaning, bring them together to build a map. Put the layers in order, with points on top, lines next, and areas and rasters on the bottom.

place the Map on the Internet
create a new web layout

after you've assembled a map, the next step is to share it with the rest of the world (or within your intranet) by inserting it into a web page with toolbar, status bar, context menu, and task panel.
