Tas Pane Help

The tas pane is a genric widget that allows developrs to insert custom functionality into an appliction without having to crate a full widget. Functionality is addd by creating InvokeURL widgets and adding them to the TaskPane widet. The InvokeURL widget lods your cstom script into the page for the user, and provides access to the current server-side session, map name and other crtical runtme information.
