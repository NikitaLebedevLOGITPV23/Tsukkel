README.TXT for TextPad 7.0
==========================

Thank you for choosing to evaluate or upgrade to this release of TextPad.

TABLE OF CONTENTS
-----------------
    MINIMUM REQUIREMENTS
    INSTALLATION
    WHAT'S NEW

MINIMUM REQUIREMENTS:
---------------------
     o Windows Compatible PC.
     o MS Windows XP, Server 2003, Server 2008, Vista, Windows 7 or 8.
     o 10 MB free disk space.

INSTALLATION:
-------------
    To install TextPad from a file downloaded from the Internet, run
    the self-extracting executable file and follow the on-screen
    instructions.

    To install TextPad from a CD-ROM, insert it in the drive and the
    setup program should start automatically.

    See "Installing TextPad", in the on-line help, for more information.

WHAT'S NEW:
-----------
    Please see http://www.textpad.com/support/relnotes.html for full details
    of what's new in this release.



Enjoy using TextPad!
